# Задание b:

## На вход подается:
- ключ "ft";
- текстовый файл f1;
- строка;

## Выходные данные:
- элементы массива структур info, подходящие пол условие;

## Позитивные тесты:
- 01 - в файле одна запись (a)
- 02 - в файле несколько записей (a)
- 03 - цена всех товаров различна (a)
- 04 - цена нескольких товаров одинакова, но их количество различно (a)
- 05 - цена и количество всех товаров одинаковы (a)
- 06 - в файле одна запись, оканчивающаяся на заданную подстроку (b)
- 07 - в файле несколько записей, оканчивающихся на заданную подстроку (b)
- 08 - имя и производитель состоят из нескольких слов, разделенных пробелами (b)
- 09 - имя и производитель состоят из нескольких слов, разделенных пробелами (a)

## Негативные тесты:
- 01 - программа вызвана без параметров;
- 02 - программа вызвана с неизвестными параметрами;
- 03 - не хватает одного или нескольких параметров;
- 04 - в конце записан лишний параметр;
- 05 - файл не сущестует
- 06 - файл пустой
- 07 - последняя запись неполная (не хватает нескольких полей)
- 08 - поля описаны в неправильном порядке
- 09 - в файле нет записей, оканчивающихся на заданную подстроку (b)
